1. Changes in github reflected to local repository
	1.1 git clone enter_the_url => clone the repo from github

2. changes on local file being reflected to github file 
	2.1 make some changes in the local file
	2.2 git checkout another_txt.txt => checkout the edited file
	2.3 git add another_txt.txt => adding to the local repo 
	2.4 get status => to check if its in your local repo created
	2.5 git commit -m "new file" => commit to the repo on local system
	2.6 git push => pushing the changes to github repo

3. undo-ing the changes done to the file in local
	3.1 git pull => pulling that file from folder
	3.2 git reset --hard => undo the recent changes
